# TempHumSense

A real-time temperature and humidity monitoring system built using embedded sensors and microcontroller technology. TempHumSense collects environmental data and displays or transmits readings for monitoring, logging, and analysis.

## Features

- Real-time temperature monitoring
- Real-time humidity monitoring
- Sensor data acquisition
- Serial output for debugging
- Lightweight embedded implementation
- Easy hardware integration

## Hardware Requirements

- ESP32 / Arduino
- DHT11 or DHT22 sensor
- USB cable
- Breadboard and jumper wires

## Software Requirements

- Arduino IDE
- Required sensor libraries
- USB drivers for board
